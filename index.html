Responsive.breadcrumbs.js is a jQuery plugin that allows you to transform non-responsive breadcrumbs into a responsive one. Useful when developing a website that is responsive and at the same time has breadcrumbs.

Basic Usage

Note: you must resize the window in order to see the plugin in action.
